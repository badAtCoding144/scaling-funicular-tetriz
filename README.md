# 1v1 Tetris Clone

A real-time 1v1 Tetris clone built with React, Node.js, Express, and Socket.io.

## Features

- Real-time multiplayer gameplay
- Responsive design
- Interactive game mechanics

## Technologies Used

- **Front-End:** React, HTML5 Canvas, CSS3
- **Back-End:** Node.js, Express.js, Socket.io
